# Guide de Survie Maths974 — Prototype Astro

Ce dépôt contient une implémentation initiale du guide de survie Maths974 en Astro.

## Démarrage

```bash
npm install
npm run dev
```

Les routes importantes :

- `/terminale/algebre/2-factoriser-une-expression/` : rendu A4
- `/terminale/algebre/2-factoriser-une-expression/a5/` : rendu A5

Le contenu est entièrement défini dans le dossier `content/` conformément à la structure décrite dans le projet.
